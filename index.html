Zoi

Write Your Own Server

Building web servers doesn't have to be complex. Zoi is an ultra-simple Zig server that teaches you the fundamentals by being transparent and understandable.

Why Write Your Own Server?

  • ✦ Deep Understanding: Learn exactly how HTTP works under the hood
  • ✦ No Magic: Every line of code is readable and purposeful
  • ✦ Performance: Zig offers C-like speed with modern safety
  • ✦ Educational: Perfect for learning systems programming

The Zoi Philosophy

Zoi strives to be a server so simple you can understand every piece just by reading the code. No hidden abstractions, no framework magic—just clean, straightforward server implementation.

By keeping the codebase minimal and focused, Zoi serves as both a functional server and an educational tool for anyone wanting to understand web server fundamentals.
